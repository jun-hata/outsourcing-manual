# リサーチ手順

## 🎯 この作業の目的
売れる可能性が高く、かつ利益が出る商品を見つけることが目的です。

## 🧰 使用ツール
- テラピーク
- eBay
- e-Ship
- Google検索
- リサーチ補助ツール

## 📋 リサーチ手順

### Step1：テラピークを開く
テラピークを開き、リサーチを開始します。

### Step2：キーワードを入力する
リサーチ対象となるキーワードを入力します。  
※キーワード選定は別ページで解説します（後で作ります）。

### Step3：販売履歴を確認する
テラピーク2を使用し、対象アイテムの販売履歴を確認します。

### Step4：販売実績をチェック
次の基準を満たすものをリサーチ対象にします。

- **90日以内に2個以上販売されている**
- 販売された価格＝相場の目安として確認する

### Step5：国内仕入れ先を検索
ツールを利用して国内検索します。  
**Google検索は特に重要**です。

### Step6：eBayの現在最安値を確認
eBay上で現在販売されている最安値を確認します。

### Step7：利益計算を行う
ツールとe-Shipを使って利益計算します。

### Step8：リストアップ基準
- **利益額が500円以上**ならリストアップ対象

## ✅ 作業完了チェック
- [ ] 販売履歴を確認した（90日で2個以上）
- [ ] 国内仕入れ先を確認した
- [ ] eBay最安値を確認した
- [ ] 利益計算をした
- [ ] 利益500円以上を確認した
